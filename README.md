# oddworks-cli

## Install

```
> npm install -g @oddnetworks/oddworks-cli
```

## Commands

```
  Usage: oddworks [cmd]


  Commands:

    generate-token   Generate a platform JWT for the channel and platform ids specified.
    token-list       List currently available JWT Tokens
    generate-uuid    Generate a mostly random UUID for use as an object id
    fetch-data       Fetch example data
    help [cmd]       display help for [cmd]

  Options:

    -h, --help  output usage information
```
